# AI-Driven Job Outreach Automation (Apache Airflow)

This project automates personalized job outreach using:
- Apache Airflow for orchestration
- Python for data processing
- LLM APIs for email generation
- AWS S3 for storage and tracking

## Architecture
- Excel job input
- LLM-powered email generation
- Automated email sending
- Application tracking and observability

## Tech Stack
- Apache Airflow
- Docker
- Python
- AWS S3
- LLM APIs
